# AI Image Generation App - Unleashing the Power of Hooks
A simple and modern web application built with **Next.js** that generates images from user prompts using the GPT-4 Image Generation API (via RapidAPI).

This project allows users to input a text prompt and instantly receive an AI-generated image based on their description. It includes a backend API route, frontend UI, constants, interfaces, and image rendering components.



## Features

- Generate images from text prompts  
- Uses GPT-4 Image Generation API (RapidAPI)  
- Includes Next.js API route under `/api/generate-image`  
- Clean UI built with React + Tailwind CSS  
- Modular file structure  
- Fully typed with TypeScript
